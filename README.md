# Wissenschaftliches Arbeiten in der Informatik – Rechenzeitvergleiche

Dieses Repository enthält Unterlagen zum Modul *SK1007 – Wissenschaftliches Arbeiten in der Informatik*, welches während des Wintersemesters 2019 an der Technischen Hochschule Mittelhessen angeboten wurde.
Verantwortlicher des Kurses war Prof. Dr. Axel Schumann.

Der Kurs setzt sich mit dem Wissenschaftsverständnis der Informatik auseinander und vermittelt die Techniken wissenschaftlichen, systematischen Untersuchens konkreter Fragestellungen und deren sorgfältige, nachvollziehbare Darstellung.
Insbesondere war das Ziel des Kurses, eine konkrete wissenschaftliche Fragestellung auf systematische und nachprüfbare Weise zu untersuchen und eine wissenschaftliche Ausarbeitung zu verfassen, in welcher die Fragestellung, das Vorgehen und die Ergebnisse in einer klaren, nachvollziehbaren Argumentation dargestellt werden.

## Wissenschaftliche Fragestellung und Inhalt

Meine [Ausarbeitung](ausarbeitung.pdf) in diesem Repository setzt sich mit der Fragestellung auseinander, wie sich verschiedene herkömmliche Ausführungsmodelle miteinander unterscheiden.
Der Fokus liegt hierbei auf dem Unterschied in der Ausführungszeit zwischen Programmiersprachen, die kompiliert, interpretiert oder auf einer Virtuellen Maschine ausgeführt werden.
Als Vertreter der verschiedenen Ausführungsmodelle werden *Haskell* (kompiliert), *Python* (interpretiert) und *Java* (Virtuelle Maschine) herangezogen.
